# Vending machine api

## Usage:

> Postman collection available in root dir.<br/>

### With docker

```console
docker-compose -f docker-compose.yml -f docker-compose-prod.yml up --build
```

### Without docker

1- install dependencies
```console
npm i
```
2- build
```console
npm run build
```
4- run server
```console
npm run up-prod
```
